index.html:整个html页面

main.js 引用在index.html页面的js文件，用来创建vue实例，公用的css和jss文件都能放在里面

app.vue 存在index.html 最外面的组件，公共的头部和尾部都能放在里面

views 存放每一个单独的主页面的

styles 存共用样式的文件夹

rouer/index.js 管理路由的文件

components 存放子组件的文件夹

assets 放静态文件的文件夹

api  管理后台请求的文件夹


用npm安装的插件可以通过 import axios from 'axios'导入

  npm i axios -S 安装
